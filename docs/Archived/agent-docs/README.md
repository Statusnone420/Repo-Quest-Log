# Archived Agent Docs

This folder keeps retired agent instruction files as historical reference without leaving them in the repository root.

Root-level agent docs are active context for their tools:

- `AGENTS.md` is the active RepoLog agent instruction file.
- `CLAUDE.md` is intentionally archived here because Claude is not the current RepoLog implementation agent.
- `GEMINI.md` is intentionally archived here because Gemini is not the current RepoLog planning agent.

RepoLog ignores archived paths by default through `.repolog.json`, so these files should not shape active workspace scope, prompt resume state, or agent-doc status. Keep them when past guidance is useful. Delete them when the project no longer needs the historical record.
